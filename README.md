#Streamix 

Streamix é uma aplicação web que simula uma rede social de vídeos e posts de texto, onde o usuário pode navegar entre telas de login, cadastro, feed, chat e perfil, com direito a modo escuro e consumo de uma API externa para popular o feed.

Projeto desenvolvido para a disciplina de Front-end, atendendo aos requisitos de HTML, CSS, JavaScript, Git/GitHub e responsividade.

#Descrição do Projeto

A Streamix tem como objetivo reproduzir a experiência de uma rede social moderna, permitindo:

- Visualizar um feed com posts/vídeos.
- Navegar entre diferentes telas (login, cadastro, recuperação de senha, feed, chat, perfil).
- Simular dados vindos de uma API externa (JSONPlaceholder).
- Alternar o tema da interface (modo claro/escuro).
- Utilizar um layout responsivo, funcionando em dispositivos móveis e desktop.

 #Tecnologias Utilizadas

- HTML5 – estrutura semântica das páginas (Login, Cadastro, Feed, Chat, Perfil, Recuperar senha).  
- CSS3 – estilização, layout responsivo e modo escuro.  
- JavaScript (Vanilla) – lógica de interação, manipulação do DOM e integração com API.  
- localStorage – simulação de persistência de dados no navegador.  
- JSONPlaceholder API – obtenção de posts externos para popular o feed.  
- Font Awesome (ícones) – ícones utilizados para enriquecer a interface.  

 #Como Rodar o Projeto Localmente:

 Pré-requisitos:
- Navegador atualizado 
- (Opcional) Editor como VS Code + extensão Live Server.  

 Passo a passo:

1. Clonar o repositório:
   *bash
•	git clone https://github.com/marinafidelis/projeto-streamix.git

2. Acessar a pasta do projeto:
•	cd projeto-streamix

3. Abrir o projeto no navegador:
•	Opção 1: clicar duas vezes no arquivo index.html
•	Opção 2 (recomendada): abrir a pasta no VS Code, clicar com o botão direito no index.html e escolher "Open with Live Server".
